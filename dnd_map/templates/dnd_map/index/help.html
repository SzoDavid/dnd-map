Help

Model fields

Name

Name of the place.

Available for: Kingdom, City, Place, Terrain

Pronunciation

If it's needed, you can specify how the name should be pronounced.

Available for: Kingdom, City, Place, Terrain

Parents

Kingdom

Select the kingdom the city is in, or select one, if the terrain appears on its map.

Available for: City, Coords

City

Select the city the place is in, or select one, if the terrain appears on its map.

Available for: Place, Coords

Terrain

Select the terrain feature you want to appear on the selected coordinates.

Available for: Coords

Type

A custom type you can define to better describe the place. For example a settlement can be a capital city, big/small or just a town.

Available for: City, Place, Terrain

Description

A short description of the place.

If it's a terrain feature and show description is not checked, people won't see it. Ideal when the player's haven't yet discovered the place.

Available for: Kingdom, City, Place, Terrain

Map

You can upload a picture of the map of the place.

Available for: Kingdom, City, Place

Coords

If the place's parent has a map, you can specify rectangles on it by its top and bottom pixel coordinates. When you click on that area on the map, it will redirect you on this place's page.

To get the coordinates open the map in an image editor where it displays the cursors coordinates (MS Paint, GIMP), move your cursor to the upper-left corner of the area, type down the coordinates, then after a comma write down the lower-right corner's coordinates. So the coordinate should be in top-left-width,top-left-height,bottom-right-width,bottom-right-height form.

NOTE: If different areas overlap, the one's that are under the other's may not be visible. For this reason, it's recommended to put these areas over the labels on the map if it's possible. If not keep it in mind that the order is kingdom > terrain > city > place where place appears on the top and kingdom on the bottom.

Available for: Kingdom, Coords

Index coords

Coordinates on the main map.

Available for: City, Place

Kingdom coords

Coordinates on the kingdom the place is in.

Available for: City, Place

City coords

Coordinates on the kingdom the place is in.

Available for: Place

Location

Select in what do you want the terrain feature to appear. Can be either index, kingdom or city.

Available for: Coords

Capital

Select if the city is the capital of the kingdom.

Available for: City

Discovered

The players will only be able to see this place if it's discovered, but you can see everything even if it's undiscovered.

Available for: Kingdom, City, Place

Show description

Similar to discovered. Hide terrain feature description from users. Ideal when the player's haven't yet discovered the place.

Available for: Terrain
