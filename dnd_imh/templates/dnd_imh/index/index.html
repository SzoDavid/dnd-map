Welcome to the Interactive Map Hub

Help your players imagine the world they are playing in!

You can add all the cities in your world, with their maps, and add links to the maps, so if someone clicks on a place, they can see all the details about it.

Also, you can group the places in hierarchic order, to make it clear which place is where.

Plan your world, without the players knowing it!

You can set if a place has been discovered yet, and if it hasn't been yet, the players won't see it. You can hide descriptions too, to keep the mystery. There is also place for notes, which are for you only.
